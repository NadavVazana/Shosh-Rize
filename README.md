## <a id="description" /> Site's Description

A site for a course that Shosh Rize's teaching, which includes babies exercises and sleeping routine guide.


## <a id="start" /> Getting Started

You can visit the site [here](https://shosh-rize.netlify.app/)

OR clone it and run it from your computer!

Head to the repository on top and clone the project or download the files.

```
git clone https://github.com/NadavVazana/Shosh-Rize
```

run this command on the terminal:
```
npm start
```

and the site will automatically open on the browser!
